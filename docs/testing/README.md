# Testing Documentation

This folder contains test reports and testing documentation for the 8th Grade Pre-Algebra platform.

## Test Reports

- **3D_BALANCE_TEST_REPORT.md** - Testing of Three.js balance visualization
- **BACK_BUTTON_TEST_REPORT.md** - Back button navigation testing
- **RETURNING_USER_TEST_REPORT.md** - User session persistence testing
- **WORD_PROBLEM_TEST_REPORT.md** - Word problem functionality testing
- **BUG_SUMMARY.md** - Known bugs and issues tracker

## Test Execution

All test scripts are located in `/tests/` directory:
- Unit tests: `/tests/unit/`
- Integration tests: `/tests/integration/`
- E2E tests: Run with Playwright
- HTML test pages: `/tests/html/`
