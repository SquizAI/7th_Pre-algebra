```

Complete Lesson 6 â†’ Streak: 5 (unchanged)
Complete Lesson 7 â†’ Streak: 5 (unchanged)

```

### 2. Completing Lesson on A-Day
```
Try to complete on A-day â†’ No streak change
Message: "Not a class day (B-day)"
Lesson still saves, but streak unaffected
```

### 3. Browser/Device Change
```
Device A: Streak 5 days (in localStorage)
Device B: Streak 0 days (new device)

Solution: Data is per-browser
Future: Sync via Supabase
```

## Performance

### Fast Operations
- Check streak status: ~1ms
- Update streak: ~2ms
- Render calendar: ~10ms (90 days)
- Show celebration: ~5ms + animation time

### Optimizations
- Calendar data cached for 5 minutes
- Confetti uses CSS animations (GPU accelerated)
- LocalStorage batched updates
- Event debouncing on rapid updates
