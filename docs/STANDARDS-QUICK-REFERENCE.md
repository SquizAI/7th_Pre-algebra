## Quick Study Guide for Each Standard

### When teaching each standard, ensure students can:

**MA.8.NSO.1.1** → Classify any number as rational or irrational  
**MA.8.NSO.1.2** → Place irrational numbers on number lines  
**MA.8.NSO.1.3** → Apply exponent rules without a calculator  
**MA.8.NSO.1.4** → Convert between standard and scientific notation  
**MA.8.NSO.1.5** → Multiply/divide in scientific notation  
**MA.8.NSO.1.6** → Interpret real-world scientific notation  
**MA.8.NSO.1.7** → Evaluate complex expressions  

**MA.8.AR.1.1** → Simplify monomial expressions using exponent rules  
**MA.8.AR.1.2** → Multiply any two binomials (FOIL)  
**MA.8.AR.1.3** → Factor expressions with common monomial factors  

**MA.8.AR.2.1** → Solve any multi-step linear equation  
**MA.8.AR.2.2** → Graph inequality solutions on number lines  
**MA.8.AR.2.3** → Solve x² = p and x³ = q  

**MA.8.AR.3.1** → Determine if relationship is proportional  
**MA.8.AR.3.2** → Find slope from any representation  
**MA.8.AR.3.3** → Write y = mx + b from any representation  
**MA.8.AR.3.4** → Graph a line from equation or description  
**MA.8.AR.3.5** → Explain slope and y-intercept meaning  

**MA.8.AR.4.1** → Check if point satisfies a system  
**MA.8.AR.4.2** → Determine number of solutions from graph  
**MA.8.AR.4.3** → Find intersection point graphically  

**MA.8.F.1.1** → Identify functions and find domain/range  
**MA.8.F.1.2** → Distinguish linear from non-linear  
**MA.8.F.1.3** → Describe where function increases/decreases  

**MA.8.GR.1.1** → Use a² + b² = c² to solve problems  
**MA.8.GR.1.2** → Find distance between two points  
**MA.8.GR.1.3** → Check if sides form triangle/right triangle  
**MA.8.GR.1.4** → Solve problems with angle relationships  
**MA.8.GR.1.5** → Find missing angles in triangles  
**MA.8.GR.1.6** → Calculate interior angles of polygons  

**MA.8.GR.2.1** → Identify transformation types  
**MA.8.GR.2.2** → Calculate scale factor  
**MA.8.GR.2.3** → Write coordinate rules for transformations  
**MA.8.GR.2.4** → Use proportions with similar triangles  

**MA.8.DP.1.1** → Create appropriate graphs for data
