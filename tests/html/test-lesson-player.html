Manual Test Checklist

Follow these steps to verify the lesson player:

  1. Start Lesson 1
    • Click "Lesson 1" above
    • Verify lesson title displays correctly
    • Verify progress shows "1 / 7"
  2. Answer Exercise 1 (Multiple Choice)
    • Click an option
    • Verify "Check" button becomes enabled
    • Click "Check"
    • Verify feedback appears (correct or incorrect)
    • Click "Continue"
  3. Test Math Problem (Exercise 4)
    • Type answer in input field
    • Click "Check"
    • Verify correct answer (x = 4)
  4. Test Incorrect Answer
    • Intentionally answer wrong
    • Verify shake animation
    • Verify hint appears
    • Verify correct answer shown
  5. Complete Full Lesson
    • Answer all 7 exercises
    • Verify completion modal appears
    • Verify XP and coins display
    • Click "Continue" to return to menu
  6. Test Keyboard Navigation
    • Use Tab to navigate options
    • Use Arrow keys in multiple choice
    • Press Enter to submit answer
  7. Test Exit Button
    • Click X in top-left
    • Verify confirmation dialog
    • Test both "OK" and "Cancel"
  8. Test Mobile Responsive
    • Open DevTools (F12)
    • Toggle device toolbar (Ctrl+Shift+M)
    • Test on iPhone/iPad sizes
    • Verify layout adjusts properly

Browser Console Tests

Open browser console (F12) and run these commands:

Check lesson progress:

LessonScheduler.getProgress()

View lesson results:

JSON.parse(localStorage.getItem('lessonResults'))

Reset progress:

LessonScheduler.resetProgress()

Launch specific lesson:

LessonScheduler.launchLesson(2)

Check current exercise type:

LessonPlayer.exercises[LessonPlayer.currentExerciseIndex]

Integration with Main App

To launch lessons from the main app, use:

From "Start This Level" button:

onclick="LessonScheduler.launchTodaysLesson()"

From skill tree node:

onclick="LessonScheduler.launchLesson(lessonNumber)"
